# NeoAssigngment

The Neo group Assignment, an app using the Question Master API for web service to fetch questions.

## Build With

- Kotlin
- Coroutines
- Jetpack compose
- MVVM Architecture
- LiveData
- Repository

## SDK levels supported

- Minimum SDK 24
- Target SDK 32
